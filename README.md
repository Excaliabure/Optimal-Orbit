# BOBSAT-1

This repo is related to the University of California Merced's AIAA Cube Satellite project.

# Motivation

For MATH 140 (Optimization), we decided to integrate the methods we have learned so far, namely Quasi-Newton and a variation of simplex method, simply to test the feasability and explore other options for optimizing BOBSAT-1 orbit.

The problem consists of optimizing 4 varaibles and 1 constraint. (see Optimizer_SZA_constraints_simplex_newton_method_20260423.m for more info). I might just write out mroe explicitly later on idk.

# Functional vs Applicable 

Within the main file, there are two sets, one is hand made optimizer using the above methods, the other is the Engineering applicable method. 

Simply for display, both methods will be present here for show, however engineering method will be the final one to be used
